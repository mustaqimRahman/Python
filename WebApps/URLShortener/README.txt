This is a URL shortener project.
In requires the following python packages:
1. Flask
2. Flask-SQLAlchemy
3. flask_wtforms
4. Cryptography

Run the app.py file to initiate the webserver. 
Go to the localhost to view the home page.
Insert a website link to be processed.
The site returns a short url.
Click on the shot url. It redirects to the original website.
